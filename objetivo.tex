\section{Objetivos}

Este trabalho tem por objetivo a implementação de um
Compilador que receberá como entrada um arquivo que contém
um código-fonte de um programa escrito em uma linguagem formal
ainda por ser definida e gerará um arquivo de saída que
contenha um programa semanticamente equivalente, porém
codificado em liguagem \emph{Assembly}.

Um compilador possui quatro partes principais, a saber, o
Analisador Léxico, o Analisador Sintático, o Analisador
Semântico e o Gerador de Código. Dessa forma, podemos
dividir o objetivo principal conforme abaixo:

\begin{itemize}
  \item Estudar as técnicas e ferramentas para a 
	      implementação do Analisador Léxico;
	\item Implementar o Analisador Léxico;
  \item Estudar as técnicas e ferramentas para a 
	      implementação do Analisador Sintático;
	\item Implementar o Analisador Sintático;
  \item Estudar as técnicas e ferramentas para a 
	      implementação do Analisador Semântico;
	\item Implementar o Analisador Semântico;
  \item Estudar as técnicas e ferramentas para a 
	      implementação do Gerador de Código;
	\item Implementar o Gerador de Código;
	\item Integrar todas as partes.
\end{itemize}
